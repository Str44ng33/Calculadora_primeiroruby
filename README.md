### Como Usar
**1. Instale as dependências**
    **Se for Ubuntu/Debian:**
    ```bash
    sudo pacman -S ruby
    sudo pacman -S git
    ```
    **Se for Arch linux**
    ```bash
      sudo apt install ruby
      sudo apt install git
      ```

**2. Pega o repósitorio:**
  ```bash
  git clone https://github.com/Str44ng33/Calculadora_primeiroruby/
  ```
**3. Execute**
```bash
cd Calculadora_primeiroruby
ruby calc.rb
```
